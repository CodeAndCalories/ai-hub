# AI Hub
Chat with ChatGPT, Claude, Gemini, Grok, and local Ollama models in one interface.
## What it is
- **Dashboard** — open source web app, works in any browser
- **Extension** — Chrome/Opera GX browser extension
## Features
- API and Native tab modes per AI
- Broadcast, Relay, and Debate modes
- Memory system with templates and auto-summarize
- Ollama support (free local models)
- Scratchpad for comparing responses
- Zero data collection — API keys never leave your browser
## Dashboard
Live at: https://codeandcalories.github.io/ai-hub/dashboard/
## Extension
Load unpacked from the /extension folder in Chrome or Opera GX.
## Privacy
All API calls go directly from your browser to each AI provider.
No server. No tracking. No data collection.
