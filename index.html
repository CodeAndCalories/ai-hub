Common questions

Is AI Hub really free?
Yes. AI Hub itself is completely free. You pay only for your own API usage directly to each provider — OpenAI, Anthropic, Google, xAI. Gemini has a free API tier with no credit card needed, so you can start for free right now.
Are my API keys safe?
Yes. Your keys never leave your browser. AI Hub has no backend server — every API call goes directly from your browser to each AI provider. We physically cannot see your keys.
Do I need a paid subscription to use AI Hub?
No. AI Hub itself is always free. You only need API keys from the AI providers you want to use. Gemini's API has a free tier with no credit card required — start there if you want to try AI Hub without spending anything.
What is Ollama?
Ollama lets you run AI models like Llama3, Mistral, and Phi3 locally on your computer — completely free and private. No API key needed. AI Hub connects to your local Ollama instance automatically.
What is Memstore?
Memstore is a persistent memory API that lets AI Hub remember things across sessions and devices. AI Hub's built-in memory resets when you close the browser — Memstore makes it permanent. Free tier available at memstore.dev.
Is there a browser extension?
Yes — AI Hub is also available as a Chrome and Opera GX extension. Same features in a popup, always one click away. Chrome Web Store listing coming soon.
